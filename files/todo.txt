Arena
[ ] Jogador => Helicopteros verdes
[ ] Inimigos => Helicopteros vermelhos
[ ] Objetos => Esferas azuis

Helicoptero
[ ] Delimitado por uma esfera virtual, no lugar do círculo
[ ] O canhão deve ser capaz de se mover para baixo [0, 45º]

Pousar e Decolar
[ ] Botão '+' => subir
[ ] Botão '-' => descer
[ ] Encostar no chão => Helicoptero pousado
[ ] Sair do chão => Helicoptero voando

Helicopteros inimigos
[ ] Cada um deve ser iniciado em uma altura diferente

Aparência do jogo
[ ] Pelo menos 2 modelos de luz
[ ] Parede texturizada
[ ] Teto texturizado
[ ] Chão texturizado

Resgate de Objetos
[ ] A esfera do jogador deve tocar a esfera do objeto
[ ] Funcionalidade para ver as esferas

Câmeras
[ ] Câmera 1 => cockpit olhando para frente (1ª pessoa)
[ ] Câmera 2 => atrás e em cima, olhando para o Helicoptero (3º pessoa)
[ ] Câmera 3 => em torno do Helicoptero, com rotação a partir do botão direito do mouse

Bônus 1
[ ] Adicionar 200 pixels a mais em y como visão permanente da câmera 1

Bônus 2
[ ] Desenhar um mapa fixo
